## About

tomato 是一个 golang 开发的批量运维工具，替代 ansible 和 salt，致力于简易使用和高性能

持续开发中。。。

## Installation

```bash
go build
```

## Settings

```bash
config.json
```

## Usage

```bash
Available Commands:
  agent       开启 tomato agent 服务
  exec        调用和执行 module
  help        Help about any command
  master      开启 tomato master 服务
```
